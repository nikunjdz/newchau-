# New Chautauqua - Big Medicine Central

Education for Democratic Revival - A modern recreation of the New Chautauqua section from nancho.net

## About

The original Chautauqua movement brought education and culture to rural America. This project revives that tradition for the digital age, exploring the roots of corporate domination and pathways to democratic renewal.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router DOM

## Getting Started

```bash
npm install
npm run dev
```
